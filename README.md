## 🚀 Getting Started  

Follow these steps to set up locally 👇  

```bash
# Clone the repository  
git clone https://github.com/srinumalapati/reactproject.git  

# Navigate into the project folder  
cd reactproject  

# Install dependencies  
npm install  

# Start the development server  
npm start
